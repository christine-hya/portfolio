FRONTEND ONLINE STORE

This project was part of a full-stack web development course at CodeSpace. The brief was to create a responsive, visually-engaging e-commerce site for handmade goods, fashion or furniture. Using Vue CLI for the shop page, I created a website for a fake fashion retailer. The user is able to add items to the cart, delete items and the total cost is calculated. I used local storage so that the data persists after reloading the page. Styling was done with Bootstrap.
